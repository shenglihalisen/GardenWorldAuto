# 花园世界自动化APP

基于Gemma端侧大模型的《我的花园世界》游戏自动化助手。

## 功能

- 🤖 **AI智能识别** - Gemma端侧大模型，实时分析游戏画面
- ⏱️ **每分钟检测** - 每60秒自动截图分析游戏状态
- 🌻 **自动收获** - 自动收获成熟花卉
- 💧 **自动浇水** - 自动为植物浇水
- 📋 **任务管理** - 自动领取任务奖励

## 下载模型

APP内置模型下载功能，打开APP后选择模型并点击下载即可。

可选模型：
- Gemma 3 270M Q8 (~125MB) - 轻量快速
- Gemma 4 E2B (~2.5GB) - 效果更好
- Gemma 4 E4B (~5GB) - 效果最好

## 构建

```bash
./gradlew assembleDebug
```

## 使用

1. 安装APP
2. 开启无障碍服务
3. 下载AI模型
4. 点击"初始化"
5. 点击"开始自动化"
